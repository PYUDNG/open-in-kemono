# Open in Kemono

This is a [Userscript](https://greasyfork.org/help/installing-user-scripts) that opens corresponding [kemono](https://kemono.cr/) page from [multiple services](#supported-services)

## Installation
You can choose one of the following to install:
- [Install from Github](../../raw/refs/heads/main/dist/open-in-kemono.user.js)
- Install from Greasyfork (Not available yet)

## Supported services
|Service|Page|Supported|
|:---:|:---:|:---:|
| Pixiv | `/artworks/123456` | ✔ |
| Pixiv | `/users/123456` | ✔ |
| Pixiv | `/novel/show.php?id=123456` | ✔ |
