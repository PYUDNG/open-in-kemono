# 在 Kemono 中打开
这是一个[用户脚本](https://greasyfork.org/help/installing-user-scripts)，可让你从[多个服务](#支持的服务)跳转到对应的 [kemono](https://kemono.cr/) 页面

本项目发布于
- [Github](https://github.com/PYUDNG/open-in-kemono)
- [Greasyfork](https://greasyfork.org/scripts/559907)
